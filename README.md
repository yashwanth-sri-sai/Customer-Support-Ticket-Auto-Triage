# Customer-Support-Ticket-Auto-Triage

## Overview

This project implements an end-to-end machine learning system to automatically classify customer support tickets into predefined categories using NLP.

## Dataset

A public IT service ticket dataset was used. Ticket text was preprocessed and categorized into issue types.

## Model

- TF-IDF vectorization
  
- Logistic Regression classifier

## Evaluation Metrics

- Accuracy
- Precision, Recall, F1-score (macro)
- Inference Latency
